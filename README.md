# Java-Scribble
Java library for Scribble(Numbers/Alphabets/Special characters) based on the input feed.

***************************

Clone project into eclipse and compile as a Java library.

This library can be used in Java web applications or Android mobile applications to process touch inputs and provide the corresponding alpha-numeric characters based on the sample provided in "/samples"
directory.


***************************

ScribbleProcessor class takes x and y coordinates to process the scribbles.
